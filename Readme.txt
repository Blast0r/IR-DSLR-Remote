List of material:
 * 3 Push- Buttons (Reset, Intervalometer, Shutterrelease/Start)
 * BatteryHolder
 * Red LED
 * Infrared LED
 * BC 337 Transistor
 * ATtiny85
 * IC-Holder
 * Base Resistor 150 Ohms
 * LED resistor 1kOhms (or even higher, the LED was very bright)
